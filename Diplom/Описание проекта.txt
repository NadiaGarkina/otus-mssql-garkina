Проект базы данных банковских кредитов для аналитической отчетности.

В учебном проекте создается база данных кредитов, выданных клиентам физическим лицам и сопутствующих данных по клиентам, платежам, просроченным кредитам и так далее.
Кредиты (таблица Credits) выдаются клиентам (таблица Clients), клиенты могут быть или не быть банкротами (таблица Clients_bankrupt), по кредитам клиенты делают платежи 
(таблица Payments),
также по платежам у кредита может возникнуть просроченная задолженность (таблица Credits_delay), и кредит может быть реструктурирован (таблица Credits_restructs),
отдельно создана справочная таблица с продуктами по кредитам (таблица Products).
Таблицы между собой имеют связи в виде внешних ключей, также создаются необходимые индексы.
Наполнение данными с помощью сайта https://www.mockaroo.com/ сделано на примере двух таблиц: [Credits] и [Clients], при этом отключены внешние ключи.
Далее предполагается, что таблицы могут наполняться данными по мере возникновения и могут быть использованы в аналитических целях 
(предполагаемый пользователь - аналитик кредитного процесса).
Предполагаемые задачи аналитика: смотреть количество кредитов, их сумму, платежи и просроченную задолженность по клиентам, по периодам, по продуктам и т.д.; 
анализировать данные по реструктуризациям, отслеживать динамику просроченной задолженности.
